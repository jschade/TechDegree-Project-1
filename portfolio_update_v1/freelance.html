What I did:

  • Set up email accounts

Challenges

I ran into a few challenges during this project. First off, it had been a few years since I had done any web or graphic design. I used this project as a way to ease myself back into HTML and CSS coding. This was also my first time ever working with a Lighbtox image gallery, so I had to take my time to make that work properly. The most challenging aspect of this project was attempting to make the contact form work with PHP, a language I had never used before and have no used since. I'm still in the process of figuring out how to make it work properly.
